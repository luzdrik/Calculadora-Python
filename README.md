# Calculadora em Python

![Status](https://img.shields.io/badge/status-em%20desenvolvimento-orange?style=for-the-badge)

Uma calculadora desenvolvida em Python como parte dos meus estudos na linguagem. O projeto começou como uma aplicação via terminal e está sendo expandido para uma interface gráfica utilizando Tkinter.

## ✨ Funcionalidades

  - Divisão

  - Potenciação
  - Radiciação

- Tratamento de entradas inválidas

## ▶️ Executando

```bash
git clone https://github.com/luzdrik/calculadora-python.git
cd calculadora-python

python calculadora.py
```

Para executar a versão gráfica:

```bash
python calculadora_gui.py
```
